## 🛠 About LoGeR

LoGeR stands for Long-Context Geometric Reconstruction with Hybrid Memory. This software helps rebuild 3D scenes from long sequences of images. It uses smart memory techniques to process more data at once without slowing down. This gives you reliable and clear reconstructions in a single application.

LoGeR is made to work on Windows computers. It runs in a normal desktop environment and does not need extra technical setup or programming.

## 📋 System Requirements

Before downloading LoGeR, check your Windows PC meets these minimum requirements: 

- Windows 10 or later (64-bit).
- At least 8 GB of RAM.  
- Intel i5 processor or equivalent, or better.
- 2 GB free disk space.
- A graphics card that supports DirectX 11 or later.

LoGeR needs a stable system to handle long sequences of images without interruptions.

## 🖥 Troubleshooting Tips

- If LoGeR does not start:
  - Check your Windows version matches system requirements.
  - Confirm your antivirus did not block the program.
  - Try running as administrator: right-click the `.exe` and select “Run as administrator.”

- If the program is slow or crashes:
  - Make sure other heavy programs are closed.
  - Check your computer has enough free RAM.
  - Restart the software or reboot your PC.

- If images fail to load:
  - Confirm the files are supported image types (JPG, PNG, BMP).
  - Ensure files are not corrupted.
